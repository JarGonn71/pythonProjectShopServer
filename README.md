# pythonReactProjectShop
Modern note taking application with Django and ReactJS/Redux

How to setup
===========
- Create a virtual env for this project
- Clone it
- `$ pip install -r requirements.txt`
- `$ cd mysite`
- `$ python manage.py runserver`
- `$ cd mysite/frontend`
- `$ yarn install`
- `$ yarn start`
